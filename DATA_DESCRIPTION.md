# Data description

## Reference Atlas

Contains public single-cell datasets used to define and visualize the
reference lineage-state space.

## Differentiation Models

Contains two independent adipogenic differentiation datasets used to
evaluate temporal and stage-dependent lineage-state transitions.

## Validated Datasets

Contains control, osteogenic induction, and previously validated
material-related datasets used for method validation.

## Novel Biomaterials

Contains TiO2-based and BTO-based biomaterial datasets analyzed as new
applications of scMeD-P.

## Displayed metrics

Depending on the page, displayed information may include:

- Os score
- Ch score
- Ad score
- MSC score
- Normalized lineage activities
- MSC ratio
- Differentiation score
- Lineage-dominant classification
- Dataset and cell-type annotations
